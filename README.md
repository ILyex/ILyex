# Importation Relevé Compteur Universelle

واجهة حديثة ومحسّنة لعمليات **Import / Export** مع ذكاء أعلى وفعالية أفضل للأحجام المختلفة.

## التحسينات الجديدة
- دعم أوسع للملفات: `CSV`, `TSV`, `TXT`, `JSON`, `XLSX`, `EXL`.
- Auto-mapping ذكي للأعمدة (يتعرف تلقائياً على أسماء الأعمدة الشائعة).
- استيراد ذكي عبر السحب والإفلات + شريط تقدم.
- تصدير مرن: كل البيانات أو البيانات المفلترة فقط.
- بحث فوري + Pagination لعرض النتائج بكفاءة.
- لوحتان بيانيتان ديناميكيتان:
  - منحنى تطور القيم حسب التاريخ.
  - مخطط دائري لتوزيع الوحدات.
- تصميم عصري باللون الأزرق وخطوط أنظف وواجهة أكثر احترافية.

## للمستخدم النهائي (Windows)
- شغّل `ImportationReleveCompteurUniverselle.exe` بالنقر المزدوج.
- تفتح الواجهة تلقائيًا: `http://127.0.0.1:5000`.

## بناء EXE محلياً
```bat
build_windows_exe.bat
```
